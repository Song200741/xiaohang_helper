# 02 校内办事流程手册
> 维护人：第24组项目小组
> 更新日期：2026-07-15
> 数据来源：郑州航空工业管理学院教务处、学生处、人事处官网

## 一、在校生常用业务
### 1.1 在校学籍证明办理
1. 线上：教务系统自助打印电子版学籍证明；
2. 线下：教务处学籍科窗口，携带本人校园一卡通；
3. 办理时间：工作日8:30-11:30，14:30-17:00 ⚠ 以学校官方最新通知为准
### 1.2 校园一卡通丢失补办
1. 先在校园公众号线上挂失冻结；
2. 携带身份证前往一卡通中心补办；
3. 补办工本费20元 ⚠ 以学校官方最新通知为准；
4. 一卡通中心地点：后勤服务一楼大厅。
### 1.3 转专业申请流程
1. 申请时间：大一下学期5月份统一开放申请通道 ✏️ 待核实；
2. 申请条件：平均绩点达到3.0以上；
3. 流程：教务系统填报申请→原学院审批→目标学院面试考核→教务处公示；
4. 结果公示结束后新学期正式转入新专业。
### 1.4 图书馆开放时间
工作日：8:00-21:30；周末：9:00-20:00 ⚠ 以学校官方最新通知为准
### 1.5 图书借阅规则
本科生单次最多借阅8本，借期30天，可线上续借一次。

## 二、教师教学行政业务
### 2.1 教师差旅报销流程
1. 出差前线上提交出差审批单；
2. 出差保留车票、住宿发票、餐饮票据；
3. 返校后在财务系统上传票据，学院负责人签字；
4. 财务处审核通过后统一打款至工资卡。
### 2.2 临时调课、停课申请
1. 线上教务教师端提交调课申请，填写调课原因、调换时间；
2. 教研室主任、教务处两级审批；
3. 审批通过后系统自动通知本班学生；
4. 紧急停课可直接联系教务处教务科临时备案。
### 2.3 教室多媒体设备报修
1. 报修渠道：后勤维修公众号、拨打后勤维修电话；
2. 报修需提供：教学楼、教室编号、故障类型；
3. 维修人员工作日24小时内上门检修。
### 2.4 科研项目申报渠道
1. 校级项目：每年3月人事科研处下发通知；
2. 省级、国家级项目：科研处统一发布申报指南；
3. 材料提交至科研管理科，统一汇总上报。

## 三、通用办事须知
1. 所有线下业务仅工作日办理，周末、法定节假日不办公；
2. 办理业务前可提前拨打对应科室电话确认所需材料；
3. 各类纸质材料均需A4纸张打印，手写材料不予受理。
