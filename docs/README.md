# 小航AI助手 - 郑州航空工业管理学院校园信息助手

## 项目简介
本项目是基于 Streamlit + 硅基流动大模型开发的校园专属AI助手，为新生、在校生、教师提供精准的校园信息查询服务，严格遵循防幻觉规则，保障信息真实可靠。

## 功能特性
- ✅ 身份分流：新生/在校生/教师三种角色，回答风格与内容精准匹配
- ✅ 本地知识库：读取 `data/` 下4份校园资料，回答基于真实信息
- ✅ 全场景异常处理：超时、断网、密钥失效、数据缺失等友好提示
- ✅ 静态兜底：底部电话黄页，API故障时仍可查阅官方电话
- ✅ 推荐问题：12个高频校园问题，一键快速提问
- ✅ 反诈提示：涉及转账场景自动提醒风险

## 快速开始
1. 克隆项目：`git clone <项目地址>`
2. 安装依赖：`pip install -r requirements.txt`
3. 配置密钥：在 `src/app.py` 中填写你的硅基流动API Key
4. 运行程序：`streamlit run src/app.py`
5. 访问页面：浏览器打开 `http://localhost:8501`
